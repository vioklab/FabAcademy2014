Epilog experiments

How to send to print

First is necessary to set up the file over Rhino. Using Command "P" on keyboard you can open the print configuration settings panel. After the configuration if your want to cut a vector or raster file, speed, power, color managing you need to configure the size of the cutting area over Rhino. After all these configurations you can send your print to the Epilog machine and you need to set up the new cutting manually:

  • First put your material inside the Epilog and secure all sides of the material with tape.
  • Then you need to set the X/Y origin pressing first the X/Y Off button and then setting manually the new origin and then press "Ajuste de Inico"
  • The you need to set up the focus of the machine pressing on "Enfocar"

Much of these steps can be done via software but for recommendations and to understand well the process we do it manually.
