This week we had to redesign the Hello Board and add some custom components to the new board.

Re-design board

I use Eagle freeware version to redesign the hello board.
To work with Fab inventory components it is necessary to download the eagle libraries:
http://academy.cba.mit.edu/2012/labs/providence/tutorials/Downloads/eagle_libraries.zip

Influenced by other Fab Academy student early projects I decided to add a "RGB led" and a simple button to my new board.

My first experience using Eagle was quite difficult until Luciano send a mail with a tutorial with some links for eagle begginers.
  1. http://www.youtube.com/watch?v=1AXwjZoyNno
  2. http://www.youtube.com/watch?v=CCTs0mNXY24

After take a look on the tutorials I follow mainly the same process as the video: first put all the components over the schematics and continue assigning all the "nets". Later into the board panel first I distribute all the components as I think it will work fine. Then I skip and create ease connections using "autorouter". It was bad idea! To many air connections and to much complicated routes. I had to look back to Hello board and try to follow the same order of the components and then add my custom components.

To create the "traces.png" file is necessary to hidde all layers except for top layer into Eagle. Then go to file> export> image and the you need to choose 500 dpi resolution and check the button Monochrome.

After created my png traces file I open it into Photoshop and I edited cutting the extra black parts on the borders and then I add change the canvas size expanding 20px for each side of the area.. I get the same file but 40px wider and higher.

Duplicating the same file and select the white border. Paint it black and then "select inverse" to paint the other area white.


When files are ready we go to the electronics production process.

What Iâ€™ve learned

The design process in eagle is quite difficult at the beginning but with some practice it turns quite easy. Some suggestion when you are designing your board is to put your components close to the corresponding connections. Then you can make a Ratsnest. This will avoid you to have some unexpected behaviors on your board and also avoid burn it.
