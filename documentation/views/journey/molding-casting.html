Molding and Casting

Design a 3D mold, machine it, and cast parts from it

Class webpage: http://academy.cba.mit.edu/classes/molding_casting/index.html

Design

This week is time to mold and cast.

To design the 3D object I used Rhino. We need to take in account the tools we plan to use to "Roughing" (remove material) and "Finishing" (focus on details):

  1. 1/8 end mill = 3.175 mm
  2. 1/16 end mill = 1.5875 mm
  3. 1/32 end mill = 0.79375
  4. 1/64 end mill = 0.396875 mm

Molding

Hard Mold

The process to use the milling machine have the same procedure as milling boards. We need to set up the machine first and then prepare the files into Fab Modules. The principal idea is to create a mold that makes molds.
For this exercise we used the machinable wax. This material is quite chip and easy to use.

Fab Modules

  1. Open Fab Modules on terminal: sudo fab
  2. Choose Modela .stl
  3. Select your 3D STL file
  4. Select your molding step: Roughing (1/8)
  5. Click Make Path
  6. Set your 0,0 point at the top-bottom-left corner
  7. Click on make rml
  8. Send the job to Modela Machine.

Once roughing process were done, we need to change to 1/32 or 1/64 end mill tool to the Fishing process. This process will remove all the steps created by the larger tool and make more refined objetct.

To send the "Finishing" process it is necessary to follow the same process as roughing process exept that is necessary to change the value of the tool diameter manually.. When ready you can send your file to mill.

Soft Mold

To make the soft mold I use the following materials and procedures:

  1. First you need to prepare all material that will be used to make the soft mold, including use gloves and glasses is it necessary.
  2. Use the spray Smooth-on Universal Mold release.. It need to be applied as 30cm long distance, two times waiting 3 minutes after each application.
  3. To create the soft Mold I use the PMCÂ®-121 - Easy To Use Urethane Rubber. This urethane rubber makes possible to cast materials as concrete, resins, plasters, and epoxies.
  4. I Mix the Wet and Dry following the instructions of the box. The mix is 1:1 relations so I decided to use volume measurement.
  5. Once you have both parts, Wet and Dry, on the mixer vessel, it is necessary to mix carefully both components without leaving waste on the borders and bottom of the vessel.
  6. Once rubber is mixed (Dry and Wet), you have to put the mixed rubber carfully and slowly on the hard mold until the material reaches the top of the mold.
  7. Next, it is necessary to hit, not so hard the hard mold, to the table to allow the air bublues goes out from the rubber. If you have a vacuum box you will get better results.
  8. After vacuum process it is necessary to leave the mold couch for 16 hours.
  9. Since my design have 2 sides I have to repeat the preocess again and wait the same time until the rubber is completly dry.

Casting

To follow the process of casting I use the following materials and procedures:

  1. As the urethane rubber mold is able to cast different materials I tried to minified the impact of the use of materials, so I decided to make the cast using only plastic: Smooth-Cast 305. This material is a liquid plastic that need to combine equal amounts of liquid parts A and B.
  2. Mix liquid plastic contents throuhly into a vassel.
  3. Clamp the two parts rubber soft mold using the union markers.
  4. Pour into rubber mold cavity and let cure.
  5. Demold solid plastic casting in 30 minutes and cast again.
