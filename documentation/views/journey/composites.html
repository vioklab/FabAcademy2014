Design

Trying to continue the same inspiration on nature I took the design I made in the Computer Aided Design class assigment and I transformed using Rhinoceros 3D to change the size of the design to make the base mold. As the composite needs to be a solid, instead using ribs (as you can see in the photo, I use a mesh object and then I transform into a solid making some little aesthetic changes. After the design process it is necessary to export the object as .STL as binary format.

Milling the mold

Materials

Shopbot

When STL file is ready we go to the Shopbot PartWorks3D and we open the file..
The process is very easy as the User Interface of the sofware is very intuitive and let the user follow the 7 steps to set up the milling process.
First is necessary to define the size and set the orientation of the mold. The next is to set the Roughing toolpath that correspond to the tool dimensions and how will be the strategy of milling defining the x and y direction.
One of the nice futures of the software is that you can simulate the milling and how the machine will remove the material to have the last result.

As the same procedure as we follow before use each machine is to check if it is completely clean and the we continue preparing the material. We use a doble side addesive tape to fix the material in the the sacrifice board on the CNC machine. Then is necessary to set up the x,y (0,0) and the z-axis home coordinates values. For z-axis we have to use the metal shit located on the top of the drill motor: put the metal shit on the fixed foam and then connect the pin to the top of the drilling circut.Using the sofware autodetect the z.
For x,y coordinates you need to check visually the point were is the best place to start the milling: use the panel control of the software to move the machine in the corresponding axes. Take care to not press the z button.

When the settings were correctly set press the green button over the Shopbot hardware. Then press return to continue when the software ask to you.
