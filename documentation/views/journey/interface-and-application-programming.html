UI - Processing

Installation

First at all you have to download the Processing IDE form the website. When installed and opened you can use very similar to Arduino IDE. The main difference is that instead loop() function, it is necessary to use draw() function. As a great open source project is very well documented.. Google it if you want to find a huge amount free code and examples.

ControlP5 library for the User Interface (UI)

To explain in a short way a library is a group of code that help you in a specific you when you are programming. I choose the controlP5 processing library. It will help us to create faster an user interface with buttons, sliders, knobs and others to control our output device. This free library is very easy to use and is very well documented as well.

To install the library controlP5 into Processing you have to copy the downloaded library zip file after uncompressed, into the your /Documents/Processing/libraries/ (MAC) folder. It this folder is not created please create one. The wiki page of processing is a good reference as well.

RGB LED Strip Library - Arduino IDE

As this LED Strip use the WS2801 chip to controll the LED Strip I use the Adafruit WS2801 library. After you have installed the library there some examples (Files/examples...) to start playing with the led.

Pay attention to check very well wich cable correspond to ground and voltage before connecting. This led runs under 5V, so is no necessary extra power source than the USB or FTDI cable (computer).

RGB LED strip state controller

The controller consist in 4 states (in reality the programming have 5 states, since starts in 'off'): 'DEFAUlT', 'RED','GREEN', 'BLUE'. This states are assigned to the the buttons on the interface and when they are pressed they send a Serial message to the board. The board receive the translated messages ('d','r','g','b') and then runs the code that is assigned for each message state.
