Experimenting with Rhino

I already had some experience with CAD software as Lightwave 3D and Maya and now I choose Rhinoceros because I have seen that very easy to use, very powerful and works very well with the Fablab Machines.

After expend a lot of time trying to get Rhino installed in my Mac computer, I feel quite frustrated.

Rhino installation and issues on Macbook Pro, 13’, 2008

Since I tried to install the last version of Rhino It return me an alert that my OS version need to be update to success with the installation. I was a little afraid to loose some of my “front end” computer configurations if I’ve have to install “Maverick”, the last mac OS, so I decided to try install “Virtual Box” and run a virtual machine based on Windows. The first installations succeed but some problem arise when I try to make the fullscreen Virtual Box window. Reinstall Virtual Box again and Windows. After the second try, 45 minutes later, apparently Windows OS needs a special driver to allow my computer graphic card takes my laptop (a little bit old) screen display resolution. So next, try to install Maverick and cross the fingers that it will not have lost on my computer configuration. At the end everything goes well!!

Finally Maverick OS is installed on my computer and Rhino Mac trial version (you can download here, any times you want), both for FREE and they are working now!

At the start I try to experiment how the program works and the UI of the software. While pass the hours playing with Rhino I tried to figure out a form that I like and might be the base of "Plant sense" installation.
These rendres are experiments and not all objects are represented with real scale measures: the last image presented in the gallery could be a possible model for the base.
