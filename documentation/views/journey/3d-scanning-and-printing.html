3D Scanning and Printing

Design and 3D print an object (small, few cm) that could not be made subtractively.

Class web page: http://academy.cba.mit.edu/classes/scanning_printing/index.html

3D print is not a simple technology and the process to get good results needs much more experimentation and learning of techniques for making additive parts out of a file.

In this fifth week I learn about 3d printing, scanning, the requirements to design a good objet to 3d print and some materials.
I specially focus in the use of kinetic scanner and keep the object to different 3d printers: Makerbot replicator 2 and HP projet 1500.

Scan Process

  1. Using Kinect camera and Skanect Software with some friend help I had scanned my body. It is a quite delicate the process of scanning with the kinect camera because if you go so much faster the software loose the point of reference.
  2. In my case I can’t finish the process of scanning because we loose the reference point. So I decided to repair it later with Rhino.
  3. Skanect is very easy to use an give a good tools to work or prepare the mesh file. Some of the useful tools that I use was the “remove extra objects” and the “fill holes” to repair the mesh.
  4. After fix the mex inside Skanect I export the file in two formats: .obj and .stl. I keep the .obj to later repair and delete the back of scanned body.

Customise and repair object Meshes

First I take the recommendation of my friend Mauro and I import my file into “Sculptris” software. Is a very powerful and easy software to play with meshes and organic forms. While I was working on my model I realise that the scanning original object was too damage on the back side of the body so to fix it will take me too many time. I prefer to make a version of a souvenir to used as key chain thats remember me a scene from the film Stars Wars when "Han solo” was freezer into a wall. :)
To make my souvenir I change to Rhino to cut the body back and use some solids to make a the figure.
Later I transform all my Polygons object to “meshes” and with the body I use a command of Rhino “repair meshes > fill holes”.
When everything is done without any possible error I export the object as .stl format.

In Rhino Files> ExportSelected > STL format > Uncheck Export Open Objects on the popup export settings window.

Makerbot

http://www.fablabbcn.org/maquinas/makerbot-replicator-2/
http://wiki.fablabbcn.org/Makerbot

I use MakerWare software to print my object. This software was created by the team of Makerbot and works pretty well on Mac OS.

  1. I have already prepared my file two allow printer start with the back position. It is necessary to lay down the object.
  2. Press the button “Add" and choose your STL file.
  3. Click on the button “move to platform” on the popup window.
  4. Move or rotate your object if it is necessary.
  5. Click on the “Make” button.
  6. Set up your preferences. In my case I set up:
    • Make it now
    • The replicator 2
    • MakerBot PLA material
    • Standard Resolution
    • Raft unchecked
    • Supports unchecked ( my object do not need supports)
    • 0.10 mm of layer height.
    • Rest Advanced options as default.
  7. Print

Time of print: about 45 minutes.
