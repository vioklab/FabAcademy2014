Fabrication Process

After design process is time to transform bits in atoms :)

The process to use the Shopbot millimachine starts with the same process as any other machine: Preapare the machine (clean, put the material, check files, etc)
In the case of the CNC we need to use screws to fix material to the base of the CNC aligning the 0,0 point to the top-left corner of the CNC machine.

  1. Material: OSB - (2.5 m, 1200m, 15 mm)(Oriented strand board)
  2. Machine: Shopbot Gantry
  3. End Milling Tool: 6 mm
  4. Software: Shopbot Pathworks

To comunicate and configure the machine cutting process we will use these software:

  1. Partworks (for lines)
  2. Partworks 3D (for reliefs)

Partworks 3 (lines)

  1. Import your file
  2. Make 2dD file to 3D file
  3. Resize document to the material board size
  4. Set the thinkness of the material (our case 14.3 mm)
  5. Check "use XY origin offset"
  6. Set units to "mm"

Configurations:

  1. Profile: Allow follow this line "means"
  2. Cutting Depths: 15.3 mm ( it is 1mm more than material thickness)
  3. Material Vectors: "on"
  4. Directions: Climb
  5. Name: Put a name as you want
  6. Tool: end mill (6mm)
  7. Cutting Params: Path Depth - 5 mm (how many material removes)
  8. Step Over: 8000 - 12000
  9. Feed Rate: 40 mm
  10. Plug Rate: 10 mm (z-axis)
  11. Toolpaths: Toolpaths list
    • Select Order: drill, inside, outside
    • Select item: (drill) and then save as template
    • Select item: (inside) and then save as template
    • Select item: (outside) and then save as template
    • Save complete projec for possible changes

Partworks 3d (reliefs)

  1. Cuts > C3 Home X > Use sensor machine to find the 0,0
  2. Load the File: first drill, second interiors and, exteriors
  3. Set spinnable "ON" - green button on the desk control
  4. Press "OK" after set the spinnable "ON"

Cautions

  1. Never leave the machine working alone. It is recommended to be present when the machine is working. In case any problem arise.
