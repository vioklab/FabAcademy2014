Step Response as input device

I choose the Step response because I have special interest to apply this input device into my final project to create an ambien sound enviroment based on this inputs. The principle of the board I create is to measure capacitance between human body and plants.

My based design is based on Jonathan Grinham design but i had to make my own eagle file because my Arduino pins design are quite diferent from my reference. Another detail is that I use 10 Mega ohms resistors insted 1 Mega ohm resitor used in Neil examples to increase the sensibility of the capacitance sensor. My objective is to work with this sensor as proximity device.

Arduino IDE code

For the Step Response input I use the Ardunino Cap Sense Library.
