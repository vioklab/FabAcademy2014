Plant Stimuli

Bio interactive touchable and auditive installation.

What will it do?

Following the first idea of interact and use plants as input devices the main propouse of Plant Stimuli installation is create a bio environment pot frame for plants that is able to react to human interactions using diferents capacitance sensing techniques: capacitance sense of plant pot environment frame and the swept frequency capacitive sensorfor the plant sense allowing people interact and touching and recognize different gestures. The main feedback that will receive the people will be music using simple auditive compositions and some messages via LCD display.

Who's done that beforehand?

There are many artistic, scientic and interactive projects using plants, light and sound but I could't find any reference about any micro enviroment to plant stimulation and monitoring.

Most of the inspiration I get was from the Disney Research Touché: Touch and Gesture Sensing for the Real World. I use DZL schematic Arduino Touché for create a similar to Disney Research board sensor board. For the starting code I found great starting tutorial example.
