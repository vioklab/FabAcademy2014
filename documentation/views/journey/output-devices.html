Xduino 1.0 and Servo Motor

For this week I want to experiment with Servos and Led Strip as output devices. First I tried to make an experiment using my common Arduino Uno, to use the breadboard and understand what is going on in the connections. Unfortunelly and without knowing the reason why it happens but when I connect the Led Strip that I had from previous days to my Xduino it burns... Why I don't really know, but I can imagine that the 8Mhz resonator and the power source going to the lights are not well.
So my next experiment was to use the servo motor that might I use to move some web camera later. The servo Motor that I use in this example is for continous rotation.

For the servo output I use this image to help myself with the connections and I connect to an external power source because the motor need between 4.5 - 6V to run and the other reason is to avoid power return to the PCB:

Materials

  1. Servo Moto - SpringRC SM-S4303R Continuous Rotation Servo
  2. Xduino 1.0
  3. 6 batteries AA of 1.5V
  4. Battery Holder (AA)
  5. Wires

Programming and Servo Arduino Library

To move the motors I use the Arduino Servo Library that is very easy to use as it is very well implemented and documented as many of the Arduino projects. I use the Sweep example servo code but I specially specify the port I am using to communicate to the servo and then check twice that all the connections are correct. After uploading the code to my board it start working as I expect.
