Plant Stimuli

Bio interactive Fabable Installation
How to make music with plants

Fab Academy 2014
Final Project Presentation

Plant Stimuli is based in a “swept frequency capacitive sensor”technology that make possible to create an symbiotical interaction between plants and human touch gestures as input device in order to create music (sound) as Output device or actuator.

With the idea of experiment with nature, art and technology I present "Plant Stimuli" as a part of a research to find the convergecy between these areas and bio capacitance sensors The "interactive pot frame" pretend to be use as a micro enviroment monitoring for different plants or biological experiments. For this intallation experiment as is used as Sound System Plant stimulation pot.

Above you can find a review presentation about how has been created "Plant Stimuli". As a long this creation was a long, hard and enjoyable process you can find more information on my journey assigment: Project Development

The design and production of “Plant Stimuli Protoype -1” was produced and can be elaborated in any local Fablab using “Digital Fabrication techniques”:Computer-Aided Design, Computer-Controller Cutting, electronics design and production, Embeded programming, Input and Output devices, networking and communications, mechanical design, and interface and application programming.

These installation was made using lasser cut wood, electronics and use fours open source softwares: Arduino IDE, Processing, Pure Dataand AU lab free MAC application.
